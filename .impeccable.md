## Target audience

台灣研究所考生(備考者)。多為大學高年級／應屆／重考生,需在報名季密集查詢校系招生資料、練歷屆考古題、追蹤招生時程。

## Use cases

裝置策略(非 mobile-first):

- **Desktop = 查詢效率**:跨校系比對招生組別、名額/錄取率、採計考科佔分;考古題依考科/學校/年度/題型檢索。
- **Phone = 快速查詢/確認**:臨時查某校系時程、某張卷題號。
- **Tablet = 閱讀/練題**:題幹、選項、解析的長文閱讀與題號導覽。

## Brand personality / tone

**Editorial Minimal** —— 沉穩、可信賴、低干擾、資訊密度高。視覺主題為
**Winslow Homer «A Basket of Clams»**(ColorLisa, TUNED):溫暖筆記本紙底上的墨藍主色,如安靜的書桌而非行銷頁。

- **字**:標題與長文閱讀用明體(自託管 GenWanMin2 TW)建立識別;UI chrome／表格用 Inter + Noto Sans TC;程式碼 Fira Code。
- **色**:墨藍主色 `primary` #566a93(已為 cream 底調暗過 WCAG AA)、暖金 `secondary` #a9944a、石棕中性 `neutral` #725435,
  皆置於筆記本紙底 `bg` #f8eeda、墨色 `text` #433222 之上;深色為「燈下夜讀」冷藍底奶油字。非霓虹、非深色發光風。
  (註:tokens.json 仍保留 violet/zinc/blue… Tailwind ramps 作為參考庫,非當前品牌色 — 勿引用。)
- **形**:統一圓角(0.375rem / 6px)、左對齊、虛線 empty state、uppercase eyebrow + tracking;**UI chrome 扁平無陰影**(黑板 board 面為唯一例外,帶極淡陰影),層次靠色調層+1px 邊框。
- **動**:低干擾,微互動 120ms、頁面轉場 220–320ms standard ease,不做彈跳/花俏動效;`prefers-reduced-motion` 全站關閉動效。

## Tech constraints

Nuxt 4 + Nuxt UI v4 + Tailwind v4(`@theme` 語意 token)+ MDC + TanStack Query;Cloudflare Pages(內容頁 prerender、API 驅動頁 client-side render);KaTeX 數學式;dark mode 經 Nuxt UI + ClientOnly 防 hydration 位移。

## Token system

三層:primitive(Tailwind 預設)→ semantic(`@theme` 的 `--text-*`/`--spacing-*`/`--leading-*`…)→ component(`:root` 的 `--bottom-bar-height`/`--safe-*` 等)。iOS 基礎:`--spacing-touch`(44px)、`--safe-*`、`100dvh`。詳見 `apps/web/DESIGN.md` 與 `docs/08-design-system.md`。
